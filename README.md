# banksoalutbk
# Platform Simulasi Bank Soal UTBK SNBT (SDGs 4)

Proyek ini adalah simulasi Ujian Tulis Berbasis Komputer (UTBK) yang dasarnya dibuat menggunakan bahasa C++. Proyek ini mendukung **Sustainable Development Goals (SDGs) poin ke-4**, yaitu Pendidikan Berkualitas, dengan menyediakan akses latihan soal gratis bagi para pelajar.

## Fitur Utama
- **7 Subtes Lengkap:**
  1. Penalaran Umum (PU)
  2. Pengetahuan & Pemahaman Umum (PPU)
  3. Pemahaman Bacaan & Menulis (PBM)
  4. Pengetahuan Kuantitatif (PK)
  5. Literasi Bahasa Indonesia
  6. Literasi Bahasa Inggris
  7. Penalaran Matematika (PM)
- **Timer Otomatis:** Setiap soal memiliki batas waktu pengerjaan 60 detik.
- **Sistem Skor:** Skor dihitung mirip sistem UTBK (Skala 1000).
- **Pembahasan:** Menampilkan kunci jawaban dan pembahasan setelah menjawab.
- **Database Soal:** Soal disimpan dalam file eksternal (`.txt`) sehingga mudah diperbarui.

## Cara Menjalankan Program

### Syarat
Pastikan komputer Anda sudah terinstall compiler C++ (seperti G++ atau MinGW).

### Langkah-langkah
1. Clone repository ini atau download sebagai ZIP.
2. Pastikan file `main.cpp` dan semua file `.txt` berada dalam **satu folder yang sama**.
3. Buka terminal/CMD di folder tersebut.
4. Compile program dengan perintah:
   ```bash
   g++ main.cpp -o aplikasi_utbk
